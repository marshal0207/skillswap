# 🔁 Skill Swap Platform

## 🚀 Overview

Skill Swap Platform is a web-based application that allows users to exchange skills with others instead of using money. Users can offer their own skills and request to learn new ones from others, creating a collaborative learning environment.

---

## 🎯 Problem Statement

Many individuals want to learn new skills but may not have the financial resources. At the same time, they possess valuable skills they can teach.

👉 This platform enables **peer-to-peer skill exchange**, making learning more accessible.

---

## 💡 Key Features

### 👤 User Features

* User Registration and Login system
* Profile creation and management
* Add and manage skills (Offered & Wanted)
* Browse other users and their skills
* Send and receive skill swap requests

### 🔄 Swap System

* Request skill exchange with other users
* Accept or reject swap requests
* Manage ongoing and completed swaps

### 🛠 Admin Features

* Manage users
* Monitor swap activities
* Maintain platform integrity
